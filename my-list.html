民雄美食 :
令人難忘的校園「食」光

Crocodile

Last updated Mar 29, 2019

1.

Do right 渡對

嘉義縣民雄鄉東榮路21號

看見媽媽的味道，吃完淚流滿面

渡對是台語「相遇」的是意思，是一間由老屋改建而成的餐廳，過去曾經是輾米廠，餐廳內擺設特別，優雅又不失復古，兩層樓的空間有好多趣味，富含巧思的佈置讓人有在時空內旅行的錯覺。 特別推薦苦戀花麵線，這是讓許多學生喚醒了想家的心情，是一道「看見媽媽的料理」，上桌時就能聞到苦茶油淡淡的香氣，上頭鋪上一顆酥炸過後的蛋，被苦茶油浸漬過後飽含香氣，蛋汁緩緩流出，配上多汁的梅花肉，交織出家溫暖的味道。

  • 必吃推薦 | 西班牙焗烤飯 + 湯

  • 電話 | ( 05 ) 226－2300

  • 營業時間 | 12：00－20：00 ( 周一、二休 )

    AT FUN Café & Brunch

    嘉義縣民雄鄉安和路10號

    精緻早午餐，文青的靜謐時光

    樂趣咖啡是民雄比較少見的精緻早午餐店，整體的裝潢走工業風，也是大家喜歡拍照打卡的景點，學生們不必跑到嘉義市中心，在民雄也能當文青。 樂趣餐點包括早午餐以及每日更換的私藏特餐，早午餐的部分提供歐姆蛋、帕尼尼、墨西哥酥餅，每日特餐則寫在牆面的黑板上。這間店的沙拉是特色，很多人不敢吃沙拉是因為生菜會有苦味，但在樂趣咖啡，沙拉採用歐式生菜，跟外面的美生菜口感不同，完全不會有苦澀感，淋上特製醬汁，咀嚼間會有淡淡香氣。

    樂趣的蛋捲是採用兩顆機能蛋所製成，濃郁的蛋香香氣四溢，蛋捲內包了會牽絲的濃郁起司、醬燒菇類和雞肉，非常健康而且很有飽足感。

  • 必點推薦 | 墨佬辣肉醬雞肉蛋捲

  • 電話 | ( 05 ) 206－1598

  • 營業時間 | 10：00－18：00 ( 周二休 )

    • 3.

    湖畔咖啡

    嘉義縣民雄鄉大學路一段168號

    美食配好景，中正的美只有你懂

    通常校園美食街總是雜亂、吵雜，湖畔咖啡不僅是隱藏在校園內的美食，雖然因為價位稍高一點，但卻是教授們喜歡用餐、喝咖啡沉思的地方，更是學生們聚會的場所，若想在午後有個靜謐、舒服的地方窩著，就來湖畔吧！ 湖畔咖啡的特色是冰滴咖啡，通常在咖啡廳看到滴咖啡都價格不匪，但在湖畔咖啡，餐點的附餐就是冰滴咖啡。冰滴咖啡的口感香濃、滑順，因為是費時萃取的，較能呈現出咖啡最佳的原味。

    餐廳空間寬敞，環境和擺設都很乾淨，用餐還可以一邊欣賞校園內的美景，湖光景色優美，窗邊的風景綠意盎然，平靜的午後，來到湖畔，綴飲著咖啡，更能認識中正的美麗。

  • 必點推薦 | 去骨烤雞腿

  • 電話 | ( 05 ) 272－1676

  • 營業時間 | 11：00－20：00
